# Hell-Cat-Windows-forms-Project Hell-Cat is a database driven windows forms application which stores all the data related to the beer brewing business and uses that information for business analysis. This Application has three interfaces namely Employee interface, Manager interface and supplier interface. An employee authorization is used to check the person using the application is authorized. After authentication the employees are directed to their specified interfaces based on their designation. Manager is directed to manager interface; Employees are directed to employee interface and suppliers are directed to supplier interface. 
The application uses a local database called CATBASE and store the data regarding Beer, Brewing method, Cumulative sales. Customer, Employee, Employee Incentives, and Sales. This data is stored without redundancy.
